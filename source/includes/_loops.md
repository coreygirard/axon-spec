# Loops

## While loops

```python
while var > 0:
    # something
```

## For loops

```python
for n in (0..3]:
    # outputs 1, 2, 3
```

```python
for n in some_list:
    # something
```

```python
for i, e in enumerate(some_list):
    # something
```

## Do While loops

```python
do:
    # something
while r != null
```
